Comandos básicos para usar no cmd:


Listar pasta e arquivos: dir

Navegando nas pastas: cd indo pra raiz cd /

Mover arquivo: mv + nome do arquivo ./nome da pasta destino

Retroceder níveis: cd ..

limpar terminal: cls

Criar: mkdir pasta

Criar arquivos: Echo hello > hello.txt criei um arquivo hello.txt e dentro terá o texto hello

Deletar: del + nome da pasta = para deletar arquivo

deletar pasta: rmdir

***GIT BASH****

mostrar caminho completo: pwd
